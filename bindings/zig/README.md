# Turso Zig Binding

Zig bindings for [TursoDB](https://turso.tech), wrapping the `turso_sdk_kit` C ABI for local database usage.

## Package Layout

```
bindings/zig/
├── build.zig          # Build script
├── README.md          # This file
└── src/
    ├── root.zig       # Root module (re-exports)
    └── ...            # Wrapper modules added incrementally
```

## Native Library Dependency

This binding wraps the C API generated by the `turso_sdk_kit` Rust crate. The header file `sdk-kit/turso.h` defines the public C API surface.

Build command:

```bash
cargo build --package turso_sdk_kit --lib
```

Linking the native library into the Zig build is planned for a later subtask.
